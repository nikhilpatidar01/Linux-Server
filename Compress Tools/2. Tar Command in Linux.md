To create a `.tar` archive of a directory:

## **4ï¸âƒ£. Extracting an Archive**
To extract a `.tar` archive:
```bash

```

```bash

```

## **5ï¸âƒ£. Creating a Compressed Archive**

```bash

```

```bash

```
