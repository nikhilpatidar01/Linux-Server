## üñ•Ô∏è **Master DNS Server Configuration**

### 1Ô∏è‚É£ Set the Hostname
```apache

```
#### Check:
```apache

```

### 2Ô∏è‚É£ Install BIND and Utilities
```apache

apt install bind9 bind9-utils -y
```

### 3Ô∏è‚É£ Configure the DNS Zones
```apache
nano /etc/bind/named.conf.local
```

#### Configuration Zone:
```apache
zone "nikhil.local" {
        type master;
        file "/etc/bind/forward.nikhil.local";
        allow-update { none; };
        allow-transfer { 192.168.112.54; };        
};

zone "patidar.local" {
        type master;
        file "/etc/bind/forward.patidar.local";
        allow-update { none; };
        allow-transfer { 192.168.112.54; };
};

zone "ai.local" {
        type master;
        file "/etc/bind/forward.ai.local";
        allow-update { none; };
        allow-transfer { 192.168.112.54; };
};
```

### 4Ô∏è‚É£ Create the Zone File
#### 4.1 Forward Zone for nikhil.local
```apache
nano /etc/bind/forward.nikhil.local
```
```apache
$TTL 1D
@       IN SOA  ns1.nikhil.local. root.nikhil.local. (
                                        20250324        ; serial
                                        3600            ; refresh
                                        1800            ; retry
                                        604800          ; expire
                                        86400 )         ; minimum
@               IN              NS      ns1.nikhil.local.
@               IN              NS      ns2.nikhil.local.

ns1             IN              A       192.168.112.53
ns2             IN              A       192.168.112.54
nikhil.local.   IN              A       192.168.112.53
www             IN              CNAME   nikhil.local.
```

### 5Ô∏è‚É£ Configure Firewall Rules (UFW)
```apache

ufw allow 53/tcp
ufw reload
```

### 6Ô∏è‚É£ Validate Configuration Check
```apache

named-checkconf /etc/bind/named.conf.local
```

### 7Ô∏è‚É£ Verifying Forward Zone  
#### Create then Verify:
```apache

```
#### **Output:**
```

OK
```

### 8Ô∏è‚É£ Start and Enable BIND

# üñ•Ô∏è **Secondary DNS Server (Slave) Configuration**  

### 1Ô∏è‚É£ Install and Configure BIND
```apache
apt update
apt install bind9 bind9-utils -y
```

### 2Ô∏è‚É£ Set the Hostname
```apache
hostnamectl set-hostname ns2.nikhil.local
```

### 3Ô∏è‚É£ Editing the Host File
```apache

```
#### Configuration Data:
```apache
127.0.0.1   localhost localhost.localdomain localhost4 localhost4.localdomain4
::1         localhost localhost.localdomain localhost6 localhost.localdomain6
192.168.112.54 ns2 ns2.nikhil.local
```

### 4Ô∏è‚É£ Configure the Zones on the Slave
```apache
nano /etc/bind/named.conf.local
```
```apache
zone "nikhil.local" {
        type slave;
        masters { 192.168.112.53; };
        file "/var/lib/bind/forward.nikhil.local";
};

zone "patidar.local" {
        type slave;
        masters { 192.168.112.53; };
        file "/var/lib/bind/forward.patidar.local";
};

zone "ai.local" {
        type slave;
        masters { 192.168.112.53; };
        file "/var/lib/bind/forward.ai.local";
};
```

### 5Ô∏è‚É£ Main File Configuration
```apache
nano /etc/bind/named.conf.options
```
```apache
options {
        listen-on port 53 { 127.0.0.1; 192.168.112.54; };    
        listen-on-v6 port 53 { ::1; };
        directory       "/var/lib/bind";
        dump-file       "/var/cache/bind/cache_dump.db";
        statistics-file "/var/cache/bind/named_stats.txt";
        allow-query     { localhost; 192.168.112.54; };
};
```

### 6Ô∏è‚É£ Configure Firewall Rules (UFW)
```apache

ufw allow 53/tcp
ufw reload
```

### 7Ô∏è‚É£ Validate Configuration Check

### 8Ô∏è‚É£ Verifying Forward Zone
