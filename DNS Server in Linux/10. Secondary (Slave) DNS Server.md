# Master and Slave DNS Server

## Server Details
```bash
Role              IP Address          Description
Master DNS        192.168.112.145     Primary DNS Server
Slave DNS         192.168.112.160     Secondary DNS Server
Domain Name       nikhil.local        Example Domain
```

# Master DNS Server Configuration
## 1. Set the Hostname
```bash
hostnamectl set-hostname ns1.nikhil.local
```
## Check
```bash
hostname
```

## 2. Install BIND and Utilities
```bash
yum install bind bind-utils -y
```

## 3. Configure the DNS Zones

```bash
nano /etc/named.rfc1912.zones
```
### Configuration Zone
```bash
zone "nikhil.local" IN {
        type master;
        file "forward.nikhil.local";
        allow-update { none; };
        allow-transfer { 192.168.1.160; };        
};

zone "patidar.local" IN {
        type master;
        file "forward.patidar.local";
        allow-update { none; };
        allow-transfer { 192.168.1.160; };
};

zone "ai.local" IN {
        type master;
        file "forward.ai.local";
        allow-update { none; };
        allow-transfer { 192.168.1.160; };
};
```

## 4. Create the Zone file
### 4.1 Forward Zone for nikhil.local
```bash
nano /var/named/forward.nikhil.local
```
```bash
$TTL 1D
@       IN SOA  ns1.nikhil.local. root.nikhil.local (
                                        20250324        ; serial
                                        3600            ; refresh
                                        1800            ; retry
                                        604800          ; expire
                                        86400 )         ; minimum
@               IN              NS      ns1.nikhil.local.
@               IN              NS      ns2.nikhil.local.

ns1             IN              A       192.168.112.145
ns2             IN              A       192.168.112.160

nikhil.local.   IN              A       192.168.112.145
www             IN              CNAME   nikhil.local.
router          IN              A       192.168.112.98
emp1            IN              A       192.168.112.200
emp2            IN              A       192.168.112.201

; Mail exchange
@               IN              MX      10 mail.nikhil.local.
mail            IN              a       192.168.112.150

; Text record for SPE
@               IN              TXT     "v=spr1 mx a ~all"

; Additional Services
ftp             IN              CNAME   nikhil.local.
dev             IN              A       192.168.112.100
fileserver      IN              A       192.168.112.105
db              IN              A       192.168.112.110
test            IN              A       192.168.112.115
vpn             IN              A       192.168.112.120
git             IN              A       192.168.112.125
webapp          IN              A       192.168.112.130
logs            IN              A       192.168.112.135

```



## 5. Main named.conf File Configuration
### add ip on port and allow query ip

```bash
nano /etc/named.conf
```
```bash
options {
        listen-on port 53 { 127.0.0.1; 192.168.112.160; };
        listen-on-v6 port 53 { ::1; };
        directory       "/var/named";
        dump-file       "/var/named/data/cache_dump.db";
        statistics-file "/var/named/data/named_stats.txt";
        memstatistics-file "/var/named/data/named_mem_stats.txt";
        secroots-file   "/var/named/data/named.secroots";
        recursing-file  "/var/named/data/named.recursing";
        allow-query     { localhost; 192.168.112.160; };
```
