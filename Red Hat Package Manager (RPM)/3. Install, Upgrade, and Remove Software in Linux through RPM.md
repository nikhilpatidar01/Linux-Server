## **ðŸ“Œ 1. Download an RPM Package (Using `wget` or `dnf`)**  

To manually download an RPM package, you can use `wget` or `dnf download`.  

ðŸ”¹ **Using `wget` to download from a direct URL:**

## **ðŸ“Œ 2. Install an RPM Package (`-i` option)**

ðŸ”¹ **Syntax:**  

ðŸ”¹ **Example:** Install **Firefox** and **VLC** from the downloaded RPM file:

## **ðŸ“Œ 3. Upgrade an Installed Package (`-U` option)**  

Use `rpm -U` to **upgrade** an existing package **or install it if missing**.

ðŸ”¹ **Syntax:**  

ðŸ”¹ **Example:** Upgrade **Firefox** and **VLC** to the latest version:

ðŸ”¹ **Syntax:**

ðŸ”¹ **Example:** Remove **Firefox** and **VLC** from the system:

## **ðŸ“Œ 5. Check if a Package is Installed (`-q` option)**  

To verify whether a package is installed, use:

ðŸ”¹ **Syntax:**

ðŸ”¹ **Example:** Check if Firefox and VLC are installed:

ðŸ“Œ **Output (if installed):**

ðŸ“Œ **Output (if NOT installed):**

âœ… **To count total installed packages:**
