## 1. Check if Apache is Installed
```bash
dpkg -l | grep apache2
```

---

## 2. Install Apache Web Server
Install Apache on all Debian-based systems:
```bash
apt install apache2 -y
```

---

## 3. Verify Apache Installation
Check package information:
```bash
apt show apache2
```
List installed files:
```bash
dpkg -L apache2
```
List configuration files:
```bash
apache2ctl -V
```
List documentation:
```bash
ls /usr/share/doc/apache2
```

---

## 4. Apache Configuration
### View Apache Configuration File
```bash
cat /etc/apache2/apache2.conf
```

### Check Modules Directory:
```bash
ls /etc/apache2/mods-enabled/
```

### Check Sites Configuration Directory:
```bash
ls /etc/apache2/sites-enabled/
ls /etc/apache2/sites-available/
```

---

## 5. Manage Apache Service
### Check Apache Status
```bash
systemctl status apache2
```

### Start Apache Service
```bash
systemctl start apache2
```

### Enable Apache on Boot
```bash
systemctl enable apache2
```

### Restart Apache
```bash
systemctl restart apache2
```

---

## 6. Network and Port Verification
### Check Open Ports
```bash
ss -nltup
```

### Check if Port 80 is open:
```bash
ss -nltup | grep :80
```

### Check if Apache is listening:
```bash
ss -ntulp | grep apache2
```

## 7. Firewall Configuration (UFW)
> Note: Kali by default doesnâ€™t have UFW enabled. Use `ufw` only if applicable.

### Check UFW Status
```bash
ufw status
```

### Enable UFW and allow Apache
```bash
ufw enable
ufw allow 'Apache'
```

### Allow Specific Ports
```bash
ufw allow 80/tcp
ufw allow 443/tcp
ufw allow 8080/tcp
```

### Reload Firewall
```bash
ufw reload
```

### List Rules
```bash
ufw status numbered
```

## 8. Apache Process and User Management
### Check Running Apache Processes
```bash
ps aux | grep apache2
```

### Find Apache User and Group
```bash
grep -i 'User\|Group' /etc/apache2/apache2.conf
```

## 9. Configure Website Files
### Go to Web Root
```bash
cd /var/www/html
```

### Create/Edit index.html
```bash
nano index.html
```

### Remove Existing File
```bash
rm -f index.html
```

### Set Ownership to Apache
```bash
chown -Rv www-data:www-data /var/www/html/
```

---

## 10. Load CSS Files (Templates)

### Download a Template
```bash
wget "https://www.free-css.com/assets/files/free-css-templates/download/page296/oxer.zip"
```

### Install unzip if not present:
```bash
apt install unzip -y
```

### Unzip Template
```bash
unzip oxer.zip
```

### Remove ZIP
```bash
rm -f oxer.zip
```

### Move Extracted Folder to Web Root
```bash
mv -v oxer /var/www/html/
```

### Set Ownership
```bash
chown -Rv www-data:www-data /var/www/html/
```

---

## 11. Enable CSS and SSI Support (Optional)

### Edit Apache Config
```bash
nano /etc/apache2/apache2.conf
```

Add at the end:
```apache
AddType text/html .shtml
AddType text/css .css
AddOutputFilter INCLUDES .shtml
```

### Enable Required Modules
```bash
a2enmod include
a2enmod mime
```

### Restart Apache
```bash
systemctl restart apache2
```
